# Mamo.site Bot

بوت Telegram لإدارة المواقع الإلكترونية، يتيح لك إضافة المواقع، البحث عنها، تعديلها، حذفها، وتصدير البيانات إلى ملف Excel.

## الميزات
- **إضافة موقع جديد**: إدخال اسم الموقع، الوصف، الفائدة، التصنيف الرئيسي، والتصنيف الفرعي.
- **البحث عن المواقع**: البحث باستخدام كلمات مفتاحية.
- **عرض النتائج**: عرض تفاصيل الموقع مع خيارات الرجوع، التعديل، أو الحذف.
- **تعديل المواقع**: تعديل اسم الموقع، الوصف، أو الفائدة.
- **حذف المواقع**: حذف موقع من القائمة.
- **تصدير البيانات**: تصدير جميع المواقع إلى ملف Excel.

## متطلبات التشغيل
- Python 3.8 أو أحدث.
- بيئة افتراضية (venv) لتثبيت المكتبات.
- توكن Telegram Bot (يمكن الحصول عليه من [BotFather](https://t.me/BotFather)).

## التثبيت
### 1. استنساخ المشروع
```bash
git clone <رابط المشروع>
cd mamo.site-bot
```

### 2. إعداد البيئة الافتراضية
```bash
python -m venv venv
source venv/bin/activate  # لنظام Linux/Mac
.\venv\Scripts\activate   # لنظام Windows
.\venv\Scripts\activate.ps1 
```

### 3. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 4. إعداد التوكن
أنشئ ملف `config.py` في المجلد الرئيسي وأضف توكن البوت الخاص بك:
```python
TOKEN = "your-bot-token-here"
```

### 5. تشغيل البوت
```bash
.\venv\Scripts\Activate.ps1
python main.py
```

## الاستخدام
1. ابدأ المحادثة مع البوت باستخدام الأمر `/start`.
2. اختر من الخيارات المتاحة:
   - **ابدأ الآن ▶️**: لإضافة موقع جديد.
   - **تصدير البيانات 📤**: لتصدير جميع المواقع إلى ملف Excel.
   - **البحث 🔍**: للبحث عن موقع.
3. عند البحث، اختر النتيجة المطلوبة لعرض التفاصيل، ثم يمكنك:
   - **رجوع ⬅️**: للعودة إلى قائمة النتائج.
   - **تعديل ✏️**: لتعديل بيانات الموقع.
   - **حذف 🗑️**: لحذف الموقع.

## هيكل المشروع
```
📂 mamo.site-bot
 ├── 📄 main.py         # الملف الرئيسي لتشغيل البوت
 ├── 📄 handlers.py     # معالجات المحادثة والأوامر
 ├── 📄 data.py         # دوال إدارة البيانات (إضافة، تعديل، حذف، بحث)
 ├── 📄 config.py       # إعدادات البوت (التوكن)
 ├── 📄 site_data.json  # تخزين بيانات المواقع
 ├── 📄 requirements.txt # المكتبات المطلوبة
```

## ملاحظات
- تأكد من اتصالك بالإنترنت أثناء تشغيل البوت.
- إذا واجهت مشكلة "Loading..."، أعد تشغيل البوت أو تحقق من إعدادات Webhook باستخدام:
  ```bash
  https://api.telegram.org/bot<your-bot-token>/getWebhookInfo
  ```
- إذا كان هناك Webhook مُعرف، قم بإزالته:
  ```bash
  https://api.telegram.org/bot<your-bot-token>/deleteWebhook
  ```
